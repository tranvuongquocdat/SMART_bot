PHẦN 2: LARK SUITE BASE
==========================================
Mục tiêu: Lấy được App ID, App Secret, Base URL (app_token)

Bước 1: Tạo tài khoản Lark Suite
- Vào https://www.larksuite.com (bản quốc tế)
- Đăng ký workspace mới (free plan đủ dùng)

Bước 2: Tạo Base (Bitable)
- Trong Lark, vào Base (hoặc Bitable)
- Tạo base mới, đặt tên "Task Manager"
- Tạo bảng "Tasks" với các cột:
    + Tên task (Text)
    + Khách hàng (Text)
    + Nội dung gốc (Text) — tin nhắn gốc CEO forward
    + Phân tích SMART (Text)
    + Trạng thái (Single Select): Mới / Đang làm / Xong / Quá hạn
    + Deadline (Date)
    + Độ ưu tiên (Single Select): Cao / Trung bình / Thấp
    + Ghi chú (Text)
- Tạo thêm bảng "Ideas" với các cột:
    + Nội dung (Text)
    + Ngày tạo (Date)
    + Tag (Text)
- Copy URL của base, dạng: https://xxx.larksuite.com/base/xxxxx
  → Phần "xxxxx" chính là app_token

Bước 3: Tạo App trên Lark Developer
- Vào https://open.larksuite.com/app
- Tạo Custom App mới
- Vào "Permissions & Scopes", thêm:
    + bitable:app (đọc/ghi Base)
- Vào "App Info", ghi lại App ID và App Secret
- Publish app (ấn "Create a version" → submit → tự approve trong workspace của mình)

- App ID: _______________
- App Secret: _______________
- Base app_token: _______________
- Table ID bảng Tasks: _______________ (xem trong URL khi mở bảng)
- Table ID bảng Ideas: _______________
